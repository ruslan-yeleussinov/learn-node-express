## How to Run

1. Clone the repo:
```bash
git clone https://github.com/your-username/learn-node-express.git
cd learn-node-express
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
node index.js
```

4. Open in browser:
```
http://localhost:3000
```
